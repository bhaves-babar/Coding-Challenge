#Coding Challenge Summary:

In this coding challenge, I completed both the backend and frontend tasks using the MERN stack(MongoDB, Express.js, React, and Node.js). For the backend, I set up a system to manage and handle data. On the frontend, I created a dashboard using **React** and Material UI (MUI) to design a clean and modern user interface.

To make the data more understandable, I integrated **Chart.js** to display important information through interactive charts. The dashboard allows users to easily view and analyze data in a visual format. The project combines both backend and frontend technologies to create a complete and user-friendly web application.# Coding-Challenge
